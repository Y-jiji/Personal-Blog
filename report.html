                                    SVM, What Is It & Why We Choose It

                                    Support vector machine (abbrev. SVM) is a linear model invented in last century. However, it is still powerful when combined with ensemble methods and is easy to implement. We employed SVM and finally find it is fast and perform well in cross-validation (only on training data).

                                    1-Layer Network with a special objective function

                                    SVM can be viewed as a single layer network with a special objective function. The objective function behaves differently on positive and negative samples and models both the 'direction' of the sample and the 'length' of the sample. Write the SVM objective in neural network fashion one can find it is a ReLU modified linear model with L2 regularization.

                                    Loss(w,b,x,y)= λ∥w∥2+[1−y(xTw+b)]+

                                    Moreover, SVM is also sutiable for regression, with minor modification on objective.

                                    Loss(w,b,x,y)= λ∥w∥2+|y−(xTw+b)|

                                    SMO algorithm instead of SGD

                                    SMO algorithm select a pair of index (i,j) in w and only change (wi,wj) in each iteration. Viewing the dual of objective function as a function of these two variable will render a closed-form solution that minimize the objective (just like simplex algorithm for linear programming). This is more efficient than SGD, since SGD always modify each wi in each iteration while the objective's converging speed (per iteration) is slower than SMO.

                                    First, refine the data (drop some columns, transform string into numbers, add some non-linear features)

                                    ​x
                                    import pandas as pd
                                    

                                    def refine(df: pd.DataFrame):
                                        df['cd_msa'].fillna('MSA/MD', inplace=True)
                                        drp_colname = ['loan_id', 'orig_chn']
                                        df.drop(columns=drp_colname, inplace=True)
                                        for c in df.columns:
                                            if df[c].dtype != 'object':
                                                continue
                                            unq = df[c].unique()
                                            dic = {unq[i]: i for i in range(len(unq))}
                                            df[c] = df[c].apply(lambda x: dic[x])
                                        df['orig_amt_p2'] = df['orig_amt'] ** 2
                                        df['orig_amt_p3'] = df['orig_amt'] ** 3
                                        df['orig_amt_p4'] = df['orig_amt'] ** 4
                                        df['default'] = df['foreclosure'] # foreclosure => default
                                        df['default'] = df['default'].apply(lambda x: 1 if x else -1)
                                        label_col_name = ['delinquent30' , 
                                                      'delinquent60',   
                                                      'delinquent90',
                                                      'foreclosure']
                                        df.drop(columns=label_col_name, inplace=True)
                                        return df

                                    Then, load the dataframe and apply refine function, split feature(x variable) and labels(y variable). 

                                    xxxxxxxxxx
                                    df = pd.read_csv('Training/mortgage_training_set.csv')
                                    df = refine(df)
                                    x = df[[col for col in df.columns
                                            if col != 'default']].to_numpy()
                                    y = df['default'].to_numpy()
                                    df = df.sample(frac=1.0)

                                    Divide the data into 201 folds (odd number, no boundary problem)

                                    xxxxxxxxxx
                                    from sklearn.model_selection import KFold
                                    

                                    SPLIT_N = 201
                                    kf = KFold(n_splits=SPLIT_N)
                                    kf.get_n_splits(x, y)
                                    folds = kf.split(x, y)

                                    Train the model. 

                                    xxxxxxxxxx
                                    from sklearn.svm import LinearSVR
                                    from sklearn.preprocessing import StandardScaler
                                    

                                    def get_model():
                                        from random import random
                                        return LinearSVR(
                                            C=0.6 + random() / 10000,
                                            max_iter=1e4,
                                            verbose=False
                                        )
                                    

                                    model_lst = []
                                    

                                    def evaluate(y, y_label, bal):
                                        # y < 0: model think the borrower will default, no gain no lose this case
                                        ...
                                        # y > 0: model think the borrower will not default
                                        gain = (y > 0) * (y_label == -1) == 1  # make money, not default
                                        lose = (y > 0) * (y_label == 1) == 1  # lose money, default
                                        return bal[gain].sum() * 0.3e-2 - bal[lose].sum() * 30e-2
                                    

                                    from tqdm import tqdm
                                    

                                    bar = tqdm(range(SPLIT_N))
                                    

                                    for tst_idx, idx in folds:
                                        std = StandardScaler()
                                        stdx = std.fit_transform(x[idx])
                                        model = get_model()
                                        weig = df['ocltv'].to_numpy()[idx]
                                        weig[y[idx] == 1] = -weig[y[idx] == 1] * 30e-2
                                        weig[y[idx] == -1] = weig[y[idx] == -1] * 0.3e-2
                                        try:
                                            model.fit(stdx, weig)
                                        except:
                                            bar.set_description('fail!')
                                            bar.update(1)
                                            continue
                                        tsty = model.predict(std.transform(x[tst_idx]))
                                        bar.update(1)
                                        bal = df['ocltv'].to_numpy()[tst_idx]
                                        if (evaluate(tsty, y[tst_idx], bal)
                                                > max(2/3 * evaluate(-y[tst_idx], y[tst_idx], bal), 0)):
                                            model_lst.append([model, std])
                                            bar.set_description('gain!')
                                        else:
                                            bar.set_description('lose!')
                                    

                                    bar.close()

                                    Run the model. (I usually saves the model, but training doesn't take too long, so run from the very beginning every time is not so bad). 

                                    xxxxxxxxxx
                                    df_ = pd.read_csv('Test/mortgage_test_set.csv')
                                    df_ = refine(df_)
                                    x = df_[[col for col in df.columns
                                            if col != 'default']].to_numpy()
                                    

                                    def model_pred(x, model_lst):
                                        y = None
                                        for model, std in model_lst:
                                            tmp_y = model.predict(std.transform(x))
                                            y = (tmp_y > 0) - 1/2 if y is None\
                                                else y + ((tmp_y > 0) - 1/2)
                                        return y
                                    

                                    y = model_pred(x, model_lst)
                                    y_label = df_['default'].to_numpy()
